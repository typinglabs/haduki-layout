# 設計

30キーで月配列を作る。

## 配置の特徴

清濁分置の配列を考えていたが、月林檎配列のシンプルな美しさに惹かれて清濁同置で作ってみることにした。

配列には、薙刀式と同様に

- 清濁同置
- 拗音同置
- 外来音同置

を満たすように配置する。濁音は月林檎のように排他的に配置して2打、拗音はハイブリッド月配列のように排他的に配置して2打（濁音の拗音は3打でOK）で打つ。

外来音については、規則的に配置できないので通常のシフト（長押し）で小書きにする。句読点も通常シフトで入力する。

## シフト機構について

ATC2025でぶな配列や月見草エキス配列の手の動きの少なさに惹かれた。これらはシフト面を複数用意することによって、単打から溢れたそこそこ頻度を多いかなをホームポジション付近に配置できるからだと考える。

なので、シフト面は複数用意する必要がある。また濁音シフトも1つだと10%程度集中してしまうので2つ用意することにする。

シフトを多く用意すると単打が減るので、月見草のように「ゃゅょ」を後置シフトに使う。もっとも頻度の低い「ゃ」を濁点シフトと兼用し（理由は後述）、「ゅ」「ょ」はシフトキーを兼用する。濁点キーが2つ必要なので、これは単打を1つ潰して用意することにする。

「ゃ」を濁点シフトと兼用する理由だが、ゃを兼用すると「ぎ じ ぢ び ぴ ぎゃ じゃ ぢゃ びゃ ぴゃ」は「ゃ」シフトではなく「゛」シフトを使わなければならなくなる。この例外の頻度は少ない方がいいので、後置シフトと兼用するのはゃが適していると考えた。

## 草案

![草案](./images/2025-12-02-souan.png)

左上が単打、右上が通常シフト、左下がゅ後置シフト、右下がょ後置シフト。

緑がシフト、黄色が濁音になるかな、赤が拗音になるかな、青が母音。

外来音の排他的配置についてだが、「あいうえおふてうとしち゛」を被らないように配置したい。母音以外は全部濁音化する文字なので、母音がそれらと被らないようにすればよい。

この例では「ふ」と「゛」が同じ位置にあるので直していきたい。
